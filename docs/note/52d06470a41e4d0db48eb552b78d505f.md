# 无法加载文件 xxxx.ps1因为在此系统中禁止运行脚本

## 问题

>  执行PowerShell脚本时提示 

![][1]



## 解决办法

> - 以管理员身份打开PowerShell
> - 输入Set-ExecutionPolicy RemoteSigned，执行策略更改
