# 使用花生壳进行公众号本地开发调试

> 每次调试都将项目打包上传至云服务器太麻烦。使用花生壳进行内网穿透，将本地ip放入白名单进行本地测试使用花生壳

<!--more-->

------

1. 下载安装花生壳
2. 新建应用类型为**HTTP**的应用，外网域名为给定的，内网主机按图填写，内网端口为程序运行端口

![本地配置][1]

3. 将本机ip加入**IP白名单**，`服务器配置`选项中，将服务器地址(URL)改为花生壳分配的**外网域名+项目地址**，提交完成即可
